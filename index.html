Powerful Features

Built for scale, security, and performance with enterprise-grade architecture

Flexible Storage

Pluggable backends including S3, Local Disk, and more. Scale from development to enterprise production seamlessly.

S3 Compatible Local Storage

Rich Metadata

PostgreSQL-powered metadata and tagging system. Query, filter, and organize files with enterprise-grade database performance.

PostgreSQL Tagging
⚙️

Dynamic Config

Real-time configuration via HashiCorp Consul. Adjust file limits, MIME types, and policies without downtime.

Consul Hot Reload

Enterprise Security

JWT authentication, HashiCorp Vault integration, and Redis-based token management. Security-first architecture.

Quick Start

Get up and running in minutes

🐳 Docker Setup

# Clone the repository
git clone https://github.com/Lumina-Enterprise-Solutions/prism-file-service.git

# Start with Docker Compose
docker-compose up -d

# Service ready at http://localhost:8080

🔧 Local Development

# Install dependencies
go mod download

# Set environment variables
cp .env.example .env

# Run the service
go run main.go
